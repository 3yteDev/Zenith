# Zenith

SA-MP / open.mp Launcher & Anti-Cheat Solution

## Stack

- API        : Rust + Axum + SQLx + Redis
- Launcher   : C# + WPF + Rust FFI
- Anti-Cheat : Rust + windows-rs
- Server     : Pawn (open.mp)
- Admin      : React + Tailwind
- Bot        : discord.js
- DB         : MySQL + Redis
- Files      : Cloudflare R2 + Backblaze B2
